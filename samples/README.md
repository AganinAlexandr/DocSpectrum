# Samples

Эта папка предназначена для будущих примеров данных и эталонных артефактов.

Планируемые материалы:

- примеры `паспорта раздела`
- примеры `результата сравнения`
- примеры структур `элементов`
- примеры структур `типовых групп элементов`

## Текущие материалы

- [element_base_v0](./element_base_v0/) - первая экспериментальная элементная база по 16 bundle `1724_25` / `1725_25`
- [comparison_results_v0](./comparison_results_v0/) - первые сравнения 8 пар одинаковых разделов
- [detailed_comparison_results_v0](./detailed_comparison_results_v0/) - детальные сравнения 8 пар с расшифровкой по признакам, страницам, таблицам и текстовым хэшам
- [detailed_comparison_results_v0_ios541](./detailed_comparison_results_v0_ios541/) - отдельный детальный проход для пары `ИОС5.4.1`

## Принцип подбора первых примеров

Первые практические примеры следует строить на небольших объектах капитального ремонта.

Желательно брать не случайные документы, а ряды похожих объектов:

- сначала 3-5 максимально близких объектов;
- затем аналогичные объекты с постепенно растущими отклонениями;
- затем другой похожий ряд, но уже с более заметными различиями внутри ряда.

Такой подход позволит проверить, насколько модель чувствительна к нарастанию различий и какие оси сравнения дают устойчивый сигнал.
